## ポイント
- TypeScript + Vue.jsで作成したモールス符号エディタです。
- モールス符号⇔英文の翻訳をリアルタイムでプレビューできます。
- モールス符号の入力はキーボードの「.」と「-」使用して入力する「キーボードモード」に加え、スペースキーを電鍵に見立てて入力する「電鍵モード」も用意！電鍵入力シミュレータとしても使えます。
<br>
